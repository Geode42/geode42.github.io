Geo Header Helper

What is this?

This is a small tool that I made to generate headings, specifically in the format asked for in my CS class.

How do I use it?

As soon as all 7 input fields are filled in, the heading will be generated and shown in the code block. From there you can simply copy-paste it into whatever file you're working on.
